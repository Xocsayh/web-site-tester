# 🌐 Web Site Tester

## 🇬🇧 English

Web Site Tester is a passive website security analysis tool.  
It scans websites and identifies potential security weaknesses without performing any attacks.

---

## 🇹🇷 Türkçe

Web Site Tester, pasif bir web sitesi güvenlik analiz aracıdır.  
Herhangi bir saldırı yapmadan web sitelerini tarar ve potansiyel güvenlik açıklarını tespit eder.

---

## 🚀 Features / Özellikler

### 🇬🇧 English
- HTTPS check
- Security header analysis
- Cookie security checks
- robots.txt, security.txt, sitemap.xml checks
- Login form analysis
- Technology detection
- Silent mode (only shows output if risk is found)
- JSON and HTML report generation

### 🇹🇷 Türkçe
- HTTPS kontrolü
- Güvenlik header analizi
- Cookie güvenlik kontrolleri
- robots.txt, security.txt, sitemap.xml kontrolü
- Login form analizi
- Teknoloji tespiti
- Sessiz mod (sadece risk varsa çıktı verir)
- JSON ve HTML rapor oluşturma

---

## ⚙️ Installation / Kurulum

### 🇬🇧 English

```bash
git clone https://github.com/Xocsayh/web-site-tester.git
cd web-site-tester
python3 -m venv .venv
source .venv/bin/activate
pip install .



--------USAGE------------------
web-site-tester --url https://example.com
web-site-tester --url https://example.com --verbose

Generate json report:
web-site-tester --url https://example.com --json report.json

Generate html report:
web-site-tester --url https://example.com --html report.html
